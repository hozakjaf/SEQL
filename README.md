# SEQL
A lightweight SQL tool

Version 1.0.0

<h2>Requirements:</h2>

Local server like xampp, mamp or wampp


<h2>Installation:</h2>


Unzip package<br />
Put it in Htdocs<br />
Change MYSQLUSER and MYSQLPASS in main.php<br />


<h2>Plugins used:</h2>

jquery-linedtextarea.js<br />
wow.js<br />
animate.css<br />
